# SleepGod
you can use this framework when you create your project.


  本工程主要是为了创建新工程后，能更快捷的开发android app，初步构想见本人脑图 http://naotu.baidu.com/file/0943e297f7f76d45cb6ac06f55637906
  使用方式，将本工程下载下来，在新建工程中加入本工程的 :sleepgod 模块，添加:sleepgod module依赖。

目前已经有的工具：
  1.request 向服务器发送请求，用法见 framework.sleep.com.org.sleepframework.activity.ExampleRequst  
    暂时用compileSdkVersion 22，com.android.support:appcompat-v7:22.2.1
